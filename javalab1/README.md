## Задание 1. Типы данных и арифметика

### Задача 1. Дробная часть
**Текст задачи:**
Реализовать метод `public double fraction(double x)`, который возвращает только дробную часть числа.
**Алгоритм решения:**
1. Выполнить явное приведение числа `x` к типу `int`, чтобы отбросить дробную часть.
2. Вычесть полученное целое значение из исходного числа `x`. Полученная разница и будет дробной частью.

### Задача 3. Букву в число
**Текст задачи:**
Преобразовать символ цифры ('0'-'9') в соответствующее целое число.
**Алгоритм решения:**
В таблице ASCII коды цифр идут последовательно. Чтобы получить число, нужно из кода текущего символа вычесть код символа '0' (выражение `x - '0'`).

### Задача 5. Двузначное число
**Текст задачи:**
Проверить, является ли переданное число двузначным.
**Алгоритм решения:**
Число считается двузначным, если оно находится в диапазоне [10, 99] (для положительных) или [-99, -10] (для отрицательных). Условия объединяются с помощью логического ИЛИ (`||`).

### Задача 7. Диапазон
**Текст задачи:**
Проверить, входит ли число в диапазон между `a` и `b` (границы могут быть перепутаны).
**Алгоритм решения:**
1. Сравнить `a` и `b`, определив среди них минимальное (`min`) и максимальное (`max`) значения.
2. Проверить вхождение числа в интервал: `num >= min && num <= max`.

### Задача 9. Равенство
**Текст задачи:**
Вернуть `true`, если все три переданных числа равны.
**Алгоритм решения:**
Использовать логическое И (`&&`) для проверки одновременного равенства: `a == b` и `b == c`.

## Задание 2. Условные операторы

### Задача 1. Модуль числа
**Текст задачи:**
Реализовать метод получения модуля числа.
**Алгоритм решения:**
Использовать условный оператор `if`. Если число меньше нуля, вернуть его со знаком минус (`-x`), иначе вернуть число без изменений.

### Задача 3. Тридцать пять
**Текст задачи:**
Вернуть `true`, если число делится на 3 или 5, но не на оба сразу.
**Алгоритм решения:**
1. Проверить исключающее условие: если число делится и на 3, и на 5 одновременно — вернуть `false`.
2. Если первое условие не выполнено, проверить обычную делимость на 3 или 5 через оператор `%`.

### Задача 5. Тройной максимум
**Текст задачи:**
Найти максимальное из трех чисел.
**Алгоритм решения:**
Принять первое число за текущий максимум. Последовательно сравнить его с остальными числами, обновляя значение максимума, если встретилось число больше текущего.

### Задача 7. Двойная сумма
**Текст задачи:**
Найти сумму двух чисел. Если сумма лежит в диапазоне [10, 19], вернуть 20.
**Алгоритм решения:**
Вычислить сумму `x + y`. С помощью `if` проверить вхождение в диапазон [10, 19] и, если условие верно, подменить результат на 20.

### Задача 9. День недели
**Текст задачи:**
По номеру вернуть название дня недели.
**Алгоритм решения:**
Использовать конструкцию `switch-case`. Для значений 1–7 возвращать названия дней, для остальных случаев предусмотреть блок `default` с сообщением об ошибке.

## Задание 3. Циклы

### Задача 1. Числа подряд
**Текст задачи:**
Вывести строку чисел от 0 до `x`.
**Алгоритм решения:**
Использовать цикл `for`, где счетчик `i` меняется от 0 до `x`. На каждой итерации добавлять текущее число и пробел в результирующую строку.

### Задача 3. Четные числа
**Текст задачи:**
Вывести все четные числа от 0 до `x`.
**Алгоритм решения:**
Использовать цикл `for` с шагом 2 (`i += 2`). Это позволит сразу перебирать четные числа (0, 2, 4...) без дополнительных проверок внутри цикла.

### Задача 5. Длина числа
**Текст задачи:**
Посчитать количество цифр в числе.
**Алгоритм решения:**
1. Если число равно 0, вернуть 1.
2. Использовать модуль числа для корректной работы с отрицательными значениями.
3. В цикле `while` делить число на 10, пока оно не станет равно 0, увеличивая счетчик на каждом шаге.

### Задача 7. Квадрат
**Текст задачи:**
Вывести квадрат из звездочек размером `x` на `x`.
**Алгоритм решения:**
Организовать вложенные циклы: внешний отвечает за строки, внутренний — за печать символов `*` в текущей строке.

### Задача 9. Правый треугольник
**Текст задачи:**
Вывести треугольник из звездочек, прижатый к правому краю.
**Алгоритм решения:**
Вложенные циклы. Внутренний цикл печатает пробел, если индекс столбца `j` меньше чем `x - 1 - i` (где `i` — номер строки), иначе печатает `*`.

## Задание 4. Массивы

### Задача 1. Поиск первого значения
**Текст задачи:**
Найти индекс первого вхождения числа `x` в массив.
**Алгоритм решения:**
Циклом `for` пройти по массиву. Как только встретится элемент, равный `x`, немедленно вернуть его индекс. Если цикл закончился — вернуть -1.

### Задача 3. Максимум по модулю
**Текст задачи:**
Найти в массиве число с максимальным абсолютным значением.
**Алгоритм решения:**
Сравнивать модули элементов (используя `Math.abs()`). При обновлении максимума сохранять само число, чтобы не потерять его оригинальный знак.

### Задача 5. Вставка массива
**Текст задачи:**
Вставить массив `ins` в массив `arr` на позицию `pos`.
**Алгоритм решения:**
1. Создать новый массив, размер которого равен сумме длин двух исходных.
2. Поэтапно скопировать: первую часть `arr` до позиции `pos`, затем весь массив `ins`, и в конце — остаток массива `arr`.

### Задача 7. Возвратный реверс
**Текст задачи:**
Вернуть новый массив, который является копией исходного в обратном порядке.
**Алгоритм решения:**
Создать новый массив. В цикле заполнять его с конца: в `i`-ю позицию нового массива записывать элемент с позиции `length - 1 - i` исходного.

### Задача 9. Все вхождения
**Текст задачи:**
Вернуть массив индексов всех вхождений числа `x`.
**Алгоритм решения:**
1. Первым проходом посчитать, сколько раз встретилось число `x`.
2. Создать массив нужного размера.
3. Вторым проходом записать все найденные индексы в новый массив.
