## Задание 1. Инкапсуляция

### Задача 7. Непустые Имена
**Текст задачи:**  
Модифицировать сущность «Имя» так, чтобы объект был неизменяемым (Immutable), и гарантировать, что хотя бы одно поле (имя, фамилия или отчество) заполнено.  
**Алгоритм решения:**  
1. Объявить все поля класса (`surname`, `personalName`, `patronymic`) как `final`.  
2. Удалить все методы-сеттеры, чтобы состояние объекта нельзя было изменить после создания.  
3. В конструкторе добавить логическую проверку: если все три аргумента равны `null`, выбрасывать исключение `IllegalArgumentException`.  
4. Это гарантирует корректность данных на этапе создания объекта.

### Задача 8. Человек меняется
**Текст задачи:**  
Внести изменения в сущность «Человек»: ограничить диапазон роста, сделать поле «Отец» неизменяемым, а имя доступным для чтения.  
**Алгоритм решения:**

2. Поле `height` сделать приватным. Реализовать сеттер `setHeight(int height)`, который проверяет, входит ли значение в диапазон от 1 до 500. Если нет — выбрасывать исключение.
